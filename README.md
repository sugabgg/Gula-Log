# Gula-Log

Gula-Log is a SocialFi application built on Canopy Network that allows users to record on-chain activities, build reputation scores, and interact with community members through a transparent activity feed.

## Features

- Activity logging on-chain
- User reputation tracking
- Community activity feed
- Wallet authentication
- Canopy Network integration

## Use Cases

- Track user contributions
- Build on-chain reputation
- Community engagement
- SocialFi analytics

## Tech Stack

- TypeScript
- React
- Canopy SDK

## Roadmap

- MVP release
- Reputation scoring system
- Activity feed improvements
- Community leaderboard
